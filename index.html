How I work..

Plan

So before I creat a site I always plan first, Planning is very important part in developing a site which you want to build. What I do in planning phase is researching what content I will put in the site and what color palette should I use to be able to enhance the beauty of website.

Design

In this phase what I do is I make a prototype in adobe illustrator or photoshop or in paper to visualize the user interface of the website so that it will be easy for me to translate it into code. In this phase I do some analyzing into the feel and look of the site.

Code

The best part is translating the design into code. In this final phase I always faced logical and problem solving because designing is not always about how you will make it beautiful, it also have some logic and problem solving to be complete.
